# W3E-Assignment2




## Layout Strategy
- The page is structured using semantic HTML elements such as `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>`.
- Followed **mobile-first approach**

---

- **Grid** is used for:
  - The main page layout
  - The image gallery  
  
- **Flexbox** is used for:
  - Navigation bars
  - Icon-and-text rows
  - Footer columns  

---

## Breakpoints Used
Breakpoint are used for Base 320px, 375px, 480px, 768px, 991px, 1200px.

---

## Limitations
- Responsive layout should have been done more acurately
- Redundancy should have been reduced
- should have organized CSS code and add comments
